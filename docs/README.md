# Swipium Docs

This directory contains public v1 documentation for the Swipium MCP server.

## Index

- [MCP Server](mcp-server.md): server setup, client configuration, and agent integration.
- [Tool Reference](tools.md): public v1 MCP tools grouped by workflow.

## Scope

Swipium v1 is simulator-only:

- Android Emulator.
- iOS Simulator.
- Local MCP stdio server.
- Local artifacts and app-map memory.

Real devices, Jira integration, external ticket workflows, cloud execution, and certification are outside the public v1 scope.
